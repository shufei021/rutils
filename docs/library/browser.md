### isChorme

> 是否为 Chorme 浏览器

```javascript
rutils.isChorme()
```

**参数：**

​ 无

**返回值：**

​ `boolean`

​ true: 是 | false: 否

**例子：**

```javascript
rutils.isChorme()
// => true or false
```
