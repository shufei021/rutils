# 浏览器&系统&环境

Browser and system and environment

# API

名称 描述
isChrome 是否Chrome浏览器
isSafari 是否是Safari浏览器
isFirefox 是否是火狐浏览器
isMobile 是否是移动端
isIos 是否是IOS
isAndroid 是否是Android
isWx 是否是微信
isWxMobile 是否是微信移动端
isWxPc 是否微信PC端
isWxWork 是否企业微信
isWxWorkMobile 是否企业微信移动端
isWxWorkPc 是否企业微信PC端

PS

以上均是函数

# 浏览器识别

# 是否Chrome浏览器

rutils.isChrome()

// 或 rutils.isChrome.is (最新版本支持.is 直接获取)

# 是否是Safari浏览器

rutils.isSafari()

// 或 rutils.isSafari.is (最新版本支持.is 直接获取)

# 是否是火狐浏览器

rutils.isFirefox()

// 或 rutils.isFirefox.is (最新版本支持.is 直接获取)

# 移动端/PC端识别

# 是否移动端

rutils.isMobile()

// 或 rutils.isMobile.is (最新版本支持.is 直接获取)

// 不是移动端 就是PC 端
